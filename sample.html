Cloths of Heaven, by W. B. Yeats

Had I the heavens' embroided cloths,
Enwrought with golden and silver light,
The blue and the dim and the dark cloths
Of night and light and the half-light,
I would spread the cloths under your feet:
But I, being poor, have only my dreams;
I have spread my dreams under your feet;
Tread softly because you tread on my dreams.

Notes on some tages

Using <emp> usually places words in italics.
Using <strong> usually places words in bold.
Make sure you finish off the emphasis by using </emp> or </strong>

To display code, use the "code" tag, as demonstrated below.
x < 3 & x > 1
e = mc2

Making Lists

Unordered Lists (Bullet Points)

Type <ul> to start an unordered list.
Place every line between <ul> and </ul> .
Finish off by typing </ul>
Here is an example:

  • cats
  • dogs
  • pigs
  • elephants

Unordered Lists (Bullet Points)

Type <ol> to start an ordered list.
Everything else is the same as for unordered lists. Here is an example:

  1. cats
  2. dogs
  3. pigs
  4. elephants
